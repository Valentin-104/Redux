### HTTP 통신

HTTP(Hypertext Transfer Protocol)는 웹에서 데이터를 주고받기 위한 프로토콜이다. HTTP는 클라이언트와 서버 간에 요청(Request)과 응답(Response)을 주고 받는 통신 프로토콜로, 주로 웹 브라우저와 웹 서버 간에 사용된다.

1. **요청 메서드 (HTTP Methods):**
   - **GET:** 리소스를 요청하기 위해 사용. 주로 데이터를 가져오는 데 사용.
   - **POST:** 리소스를 생성하기 위해 사용. 주로 데이터를 서버로 전송하는 데 사용.
   - **PUT:** 리소스를 업데이트하기 위해 사용. 전체 리소스를 업데이트하는 데 사용.
   - **PATCH:** 리소스의 일부를 업데이트하기 위해 사용.
   - **DELETE:** 리소스를 삭제하기 위해 사용.

2. **헤더 (Headers):**
   - 요청과 응답에 대한 부가적인 정보를 포함하는 부분. 예를 들어, 인증 정보, 캐싱 설정, 인코딩 방식 등을 포함할 수 있음.

3. **바디 (Body):**
   - 요청이나 응답에 대한 본문 데이터를 포함. 주로 POST나 PUT 요청에서 사용되며, JSON 또는 XML 형식으로 데이터를 전송할 수 있음.
