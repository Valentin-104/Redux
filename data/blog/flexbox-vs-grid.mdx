CSS Flexbox(Flexible Box)와 CSS Grid는 웹 페이지의 레이아웃을 디자인하기 위해 사용되는 CSS 레이아웃 시스템이다.

**Flexbox**
  - **일차원 레이아웃**: 주로 단일 축(행 또는 열)을 따라 요소를 배치하는 데 최적화되어 있다. 이는 작은 구성 요소나 앱 내부의 일부 영역(예: 네비게이션 바, 아이템 리스트)을 배치할 때 유용하다.
  - **유연성**: Flexbox의 이름에서 알 수 있듯이, 유연성이 핵심이다. 컨테이너의 크기가 변경되더라도, 내부 요소들은 설정에 따라 유연하게 크기 조정과 재배치가 가능하다.

**Grid**
  - **이차원 레이아웃**: 행과 열을 모두 고려하는 이차원 레이아웃을 생성하는 데 최적화되어 있다. 이는 페이지 전체의 레이아웃이나 큰 섹션의 레이아웃을 설계할 때 유용하다.
  - **정교한 레이아웃 구성**: 웹 페이지를 격자(grid)로 생각하고, 이 격자 위에 웹 요소를 배치한다. 이는 복잡한 디자인과 레이아웃을 효율적으로 구현할 수 있게 해준다.
