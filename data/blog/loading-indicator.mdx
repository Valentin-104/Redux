### Animation using Tailwind CSS

Tailwind CSS에는 CSS animation을 위한 여러가지 utility class가 있다. 그 중 한 가지를 이용해서 loading spinner를 만들어, 로딩 중인 상태를 나타내고 싶었다.

`animate-spin`이라는 class를 찾았는데, 이는 다음과 같은 CSS 속성을 가진다. 


```css
animation: spin 1s linear infinite;

@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
```

즉, 1초마다 360도 회전하는 animation인 것이다. 이를 가지고 loading spinner를 표현하는 `loader`라는 `className`을 만들고, 컴포넌트 `Loader`애 적용했다. 

```css
  .loader {
    @apply border-4 border-orange-300 border-t-4 border-t-orange-500 rounded-full animate-spin;
  }
```

```tsx:components/loader.tsx
type LoaderProps = {
  width?: string;
  height?: string;
};

export default function Loader({
  width = 'w-10',
  height = 'h-10',
}: LoaderProps) {
  return <div className={`loader ${width} ${height}`}></div>;
}
```

`Loader`의 크기는 상황에 따라 바꾸고 싶어서, 높이와 너비를 `prop`으로 전달하고, 전달하지 않으면 기본 40px로 설정했다.
